01 — About

About Me

I'm a full-stack software engineer focused on architecting end-to-end enterprise systems and building functional, high-performance web applications. Utilizing a robust backend toolkit of Python, PHP, SQL, and SAP ABAP alongside modern frontend technologies like JavaScript, React, Next.js, and Tailwind CSS, I design and deploy actual production tools that optimize workflows and scale technical infrastructure. My experience spans managing complex data conversions and large-scale system rollouts for multinational companies, startups, and MSMEs, as well as leading engineering teams and technical bootcamps.

Lately, my work centers heavily on advanced AI integration, moving past simple implementations to build deeply intelligent workflows and custom software solutions. I specialize in developing production-ready Retrieval-Augmented Generation (RAG) platforms, context models, and AI automation workflows using tools like Claude Code to streamline enterprise systems. A major focus of my current work is creating and integrating LLM Wikis, transforming static repository data into intelligent, structured knowledge spaces that language models can interact with seamlessly to optimize software development.
